# CustomBows

### Currently in development, may not work as intended

This plugin provides a way to have custom bows on your server.

Bow Types:

- Shotgun
- Sniper*
- Minigun*
- Rocket Launcher*

(* = not implemented yet)

The bows require player's not being able to rename their bows as it will 'break' them.
